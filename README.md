# Fake Cash - iOS App

Pixel-perfect SwiftUI replica of the Cash App Money screen.

## Requirements
- Xcode 15 or later
- macOS 13+
- iOS 16+ deployment target

## How to Open & Run

1. Unzip `FakeCash.zip`
2. Double-click `FakeCash.xcodeproj`
3. In Xcode → **Signing & Capabilities** → set your Team
4. Select iPhone Simulator (or your device) in the toolbar
5. Press **Cmd+R** to build and run

## Features
- Full scroll up/down matching screenshots exactly
- Dark header with card chip + cashtag
- White balance card with large amount display
- Add money / Withdraw buttons
- Green status banner
- "More for you" section (Bitcoin, Paychecks, Savings, Stocks, Pools)
- "Add money" section (Direct deposit, Wire, Paper money, Check, Auto reload)
- FDIC disclosure text
- Bottom tab bar

## Editing Values (In-App)
Tap the **globe avatar** (top right corner) → Edit Account sheet:
- Balance amount
- Cashtag / card display name
- Card last 4 digits
- Account last 4 digits
- Profile emoji
- Green status text
- Notification badge count
